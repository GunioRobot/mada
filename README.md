### Mada

Mada is a set of Scala objects:

- Compare

    Strict weak ordering

- Functions

    Utility methods for functions

- Iterators

    Utility methods for Iterator type

- Peg

    Lightweight PEG parser combinators

- Stl

    Port of C++ STL

- Vector

    Structurally-unmodifiable writable array
